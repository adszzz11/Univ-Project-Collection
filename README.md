# 프로젝트 소개

YJCoffee는 C# 언어 기반으로 만들어진 간단한 시스템입니다.<br />
사용자가 커피를 구매하면, 카페에서 주문을 받아 커피를 만드는 프로세스를 간단히 표현했습니다.

# 구조

1. 구성원과 역할
 - 이상민(조장) : 프로젝트 관리, 데이터베이스 연동, 코딩
 - 최익준 : 코딩, 구조 관리
 - 배성현 : 디자인, 코딩
 - 도성경 : 코딩, 알고리즘 분석
 - 최재욱 : 코딩, 성능 테스트

2. 기술 분석
* UI : C#의 WinForm을 사용하여 UI 구현.
* DataBase : MySQL을 사용하여 데이터베이스 구현.
